Windows:
  1. Double-click: launcher.bat
  2. Setup runs automatically on first launch
  3. App opens in a native window

macOS / Linux:
  1. Open Terminal in the app folder
  2. Run: sh launcher.sh
  3. Setup runs automatically on first launch
  4. App opens in a native window

 KEY IMPROVEMENTS


 Single entry point for all platforms (launcher.bat / launcher.sh)
 Auto-detecting setup (only runs on first launch)
 Automatic OS detection (Windows/macOS/Linux)
 Platform-specific dependencies (PyWebView extras)
 Clear error messages & helpful troubleshooting
 Data directory auto-creation
 Ollama auto-detection
 Modern installer experience
 Ready for standalone .exe/.app distribution
 100% backward compatible (old scripts still work)





Old Installation Flow:
 run.bat
 sh run.sh

New Installation Flow:
  Windows:  launcher.bat (setup auto-runs)
  macOS:    sh launcher.sh (setup auto-runs)

All platform-specific handling is now automatic!





Old scripts still work:
 setup.py (kept for reference)  
 run.bat  
 run.sh  

But users should use the new system:
 launcher.bat (Windows)  
 launcher.sh (macOS/Linux)  





Option 1: Source Distribution (Easy)
 double-click launcher.bat or sh launcher.sh
  Requires: Python 3.9+
  Size: ~10-15 MB

Option 2: Standalone Installers (Professional)
  Windows: SuperheroForge. double-click to installexe 
  macOS:   superhero-forge. drag to Applicationsdmg 
  Linux:   superhero- chmod +x && runforge 
  Requires: Nothing (Python included)
  Size: ~120-180 MB per platform
